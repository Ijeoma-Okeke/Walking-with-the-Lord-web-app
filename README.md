This is a web application for believers everywhere in the world to have access to various daily devotionals. 
This allows them to read and enjoy more than one devotional per time, instead of being stuck with one devotional. The web application refreshes each day and provides insight into different themes as a reader may need it.
For now, it is a project in development, but it will be updated to fit what everyone needs. 
It has devotionals for pentecostals, protestants, catholics, anglicans, and other traditions within the Christian faith.
It was built using HTML5, CSS3, and Vanilla Javascript. At this point, it doesn't need a database.
